USWDS System colors

USWDS system color tokens are a complete palette of color tokens put together by USWDS. However, because FPAC colors do not match colors in the USWDS system, these colors are provided for reference only.

Additional color guidance

  • FPAC Theme colors: The pallet of colors for USDA projects.
  • FPAC State colors: The colors for warnings, errors, and other states.
  • USWDS System colors: The underlying palette of color tokens from USWDS.

System colors

FPAC does not use these colors in its design system. They are provided here for reference only. To use them in your code, paste the hex values directly — the corresponding design tokens are not included in FPAC's SCSS/CSS.
