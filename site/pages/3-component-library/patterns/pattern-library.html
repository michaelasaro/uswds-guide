Component Library

The FPAC Component Library provides standard components for the design and development of FPAC websites. Those components have been combined into common patterns and templates to ensure a consistent look and feel across the FPAC portfolio.

Library Sections

  • Components are the basic building blocks of the design system.
  • Patterns combine multiple components to solve common user tasks or interface needs.
  • Templates arrange components and patterns into full-page layouts to address specific use cases.

Also check out our FSA Comparison Table that compares the availability of FPAC and FSA components.



While we continue to build out this site, cards with the icon link to the official USWDS documentation for that component, pattern, or template.


Patterns


  • Establish trust

    Help users feel confident that your site is official, secure, and trustworthy.

  • Keep a record

    Provide users with ways to save, print, or download records of their activity.

  • Language selector

    Language selectors allow users to choose their preferred language.

  • Select language

    Help users select their language for a better experience.

  • User profile

    Display and manage user information and account settings.
