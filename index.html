USWDS Starter

USWDS Starter Template

A clean, forkable starter template built on the U.S. Web Design System (USWDS). Everything you need to launch an accessible, mobile-friendly site.

Getting Started

  1. Install dependencies 
    npm install
  2. Build assets 
    npm run build
  3. Start development server 
    npm run serve

What's Included

  • USWDS 3.13.0 — The complete U.S. Web Design System
  • SCSS tokens — Customizable design tokens in src/styles/tokens/
  • Fonts — Public Sans, Merriweather, Source Sans Pro, Roboto Mono
  • Build system — Gulp-based compilation with USWDS Compile

Required Files

Include these in your HTML:

<!-- CSS -->
<link rel="stylesheet" href="dist/uswds/css/uswds.min.css">
<link rel="stylesheet" href="dist/fpac-style/styles/fpac-design-system.css">

<!-- JavaScript (before closing body tag) -->
<script src="dist/uswds/js/uswds.min.js"></script>

Project Structure

uswds-starter/
├── src/
│   ├── styles/        # SCSS source files
│   │   ├── tokens/    # Design tokens (colors, typography, etc.)
│   │   └── index.scss # Main entry point
│   ├── fonts/         # Web fonts
│   └── images/        # Logo and image assets
├── dist/              # Compiled output (generated)
├── gulpfile.js        # Build configuration
├── package.json       # Dependencies and scripts
└── index.html         # This page

Customization

Modify the design tokens in src/styles/tokens/ to customize colors, typography, spacing, and more. Then run npm run build to recompile.
